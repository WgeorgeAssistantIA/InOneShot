# InOneShot

Appli desktop de **publipostage PDF** : dépose un modèle PDF + un fichier Excel/CSV, place tes champs, génère un PDF par ligne en un clic.

- Site : https://inoneshot.fr
- Téléchargements : voir l'onglet [Releases](https://github.com/WgeorgeAssistantIA/InOneShot/releases)
- Microsoft Store : https://apps.microsoft.com/detail/9PPBQSM1MFZ2
